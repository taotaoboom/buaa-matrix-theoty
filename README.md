# 北京航空航天大学矩阵理论复习资料
本人是2025级秋季学院路校区王磊老师班的学生，在此将本人本学期整理+收集--->过滤后的复习资料开源，方便后来者复习。此仓库包括本人依据王磊老师的教材整理的复习笔记，往年真题；王磊老师撰写的教材扫描版和矩阵理论考试大纲。

## 关于2025年期末考试
刚考完期末考试，本人只做了2023-2025年春的真题，个人感觉2025年秋季这一次考试是比较难的，也可能我水平较差😢。\
王老师执行力也太强了，下午3：20考完试，晚上11点已经判完卷了，还在群里一直与同学们激情互动到00:38分，这老师能处！

## 关于笔记的使用
教材推荐使用王磊老师的版本，我读下来总体感觉还是逻辑清楚，解释清晰地。但是教材内容太多太杂，就会导致看教材重点不突出，所以可以看我整理的笔记，里边顺便也把教材每章的课后习题给列了出来并给出解析（仅限于考试范围内的），经过了Gemini-3的严格审核😉。另外，教材一共有5章，考试范围只在前四章，第五章不考。\
一句话总结：如果你只是期末复习，可以直接进入我的笔记的对应章节进行学习，基本覆盖全部要考的知识点。如果你想要进行修改，可以在直接使用latex文件进行修改。

## 关于往年真题的使用
这里附上王磊老师的原话：大家没必要去做往年题目，因为每次出题都换老师，出题风格和重点都不一样，不如关注**我给大家的复习重点**。不得不感叹，老师的话真的在2025秋学期应验了😂\
所以，如果选修矩阵仅仅是为了修够学分，在这里推荐大家去选王磊老师的课程，考试前会给划重点（我已经整理为图片，可以在期末周当做桌面壁纸）。当然，赵迪老师的口碑一向很好，在他的课上或许能学到一些对于写论文真正有用的数学。但是本人只是为了修学分，就选了王磊老师的‘免听不免考（不用去上课但是要考试）’班级。
